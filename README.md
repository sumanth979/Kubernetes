## Working with Minikube
##### Start the Minikube using virtualbox driver.
```bash
minikube start --driver=virtualbox
```

##### To check the status of the minikube
```bash
minikube status
```

##### To stop the running minikube
```bash
minikube stop
```

##### To cleanup the local state of minikube
```bash
minikube delete
```
